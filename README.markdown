## Installation
Clone repo
```
$ git clone https://github.com/walpurgisnatch/pero.git
```
After it add path to pero.asd in your asdf registry 
```
(push "/path/to/pero" asdf:*central-registry*)
(ql:quickload pero)
```

### With roswell
```
$ ros install walpurgisnatch/pero
```
